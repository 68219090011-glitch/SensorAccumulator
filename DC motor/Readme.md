# LEVB6 DCMOTOR
![DCMOTOR](https://abacasstorageaccnt.blob.core.windows.net/cirkit/e2e2c934-b375-45fc-834c-534243cbf361.png "DCMOTOR")

> มอเตอร์เกียร์กระแสตรง (DC gear motor) คือมอเตอร์ไฟฟ้าที่แปลงพลังงานไฟฟ้ากระแสตรง (DC) เป็นพลังงานกล มีกลไกเกียร์ในตัวที่ช่วยลดความเร็วรอบของมอเตอร์พร้อมกับเพิ่มแรงบิด ทำให้เหมาะสำหรับงานที่ต้องการการควบคุมความเร็วรอบที่แม่นยำและแรงบิดสูง มอเตอร์เกียร์กระแสตรงทำงานได้อย่างมีประสิทธิภาพในช่วงแรงดันไฟฟ้า 3 ถึง 6 โวลต์ ทำให้เหมาะสำหรับงานที่ต้องการกำลังไฟต่ำ

## การใช้งานและกรณีตัวอย่างทั่วไป
- วิทยาการหุ่นยนต์: การขับเคลื่อนล้อ แขน หรือชิ้นส่วนเคลื่อนที่อื่นๆ
- ระบบอัตโนมัติ: สายพานลำเลียง, ตัวขับเคลื่อนขนาดเล็ก หรือกลไกต่างๆ
- งานประดิษฐ์ DIY: โมเดลรถยนต์ เรือ หรือสิ่งประดิษฐ์อื่นๆ ที่ผู้ชื่นชอบงานอดิเรกสามารถทำได้
- วัตถุประสงค์ทางการศึกษา: เพื่อสาธิตหลักการควบคุมมอเตอร์และการลดเกียร์

# ข้อกำหนดทางเทคนิค
ด้านล่างนี้คือรายละเอียดทางเทคนิคที่สำคัญของมอเตอร์เกียร์ DC (3-6V):

> พารามิเตอร์	ค่า
แรงดันไฟฟ้าใช้งาน	3V ถึง 6V
แรงบิดที่กำหนด	0.5 กก.·ซม. ถึง 1.5 กก.·ซม. (แตกต่างกันไป)
ความเร็วขณะไม่มีโหลด	100 รอบต่อนาที ถึง 300 รอบต่อนาที (อาจแตกต่างกันไป)
กระแสหยุดนิ่ง	~1.2A (ที่ 6V)
อัตราทดเกียร์	โดยทั่วไปคือ 1:48 หรือ 1:120
เส้นผ่านศูนย์กลางเพลา	3 มม.
ขนาดมอเตอร์	ขนาดประมาณ 25 มม. x 20 มม. x 15 มม.
น้ำหนัก	~30 กรัม

## การกำหนดค่าและคำอธิบายของพิน
มอเตอร์เกียร์ DC โดยทั่วไปจะมีขั้วต่อไฟฟ้าสองขั้ว:

เข็มหมุด	คำอธิบาย
-	ขั้วบวกสำหรับป้อนไฟ (3-6V DC)
-	ขั้วลบสำหรับต่อลงดิน
