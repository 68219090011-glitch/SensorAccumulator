4x4 Keypad


คีย์แพด 4×4 เป็นคีย์แพดแบบเมทริกซ์ที่ประกอบด้วยปุ่มทั้งหมด 16 ปุ่ม จัดเรียงเป็น 4 แถว และ 4 คอลัมน์ นิยมใช้สำหรับรับข้อมูลจากผู้ใช้ในอุปกรณ์อิเล็กทรอนิกส์ต่าง ๆ ผู้ใช้สามารถป้อนข้อมูลตัวเลขหรือตัวอักษรได้ และมักถูกนำไปใช้ในงานอย่างเช่น ระบบรักษาความปลอดภัย เครื่องคิดเลข และระบบฝังตัว (Embedded Systems) ด้วยขนาดที่กะทัดรัดและการเชื่อมต่อที่ทำได้ง่าย จึงเป็นอุปกรณ์ที่ได้รับความนิยมสำหรับโปรเจกต์ที่ต้องการการโต้ตอบกับผู้ใช้
